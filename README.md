# ML DataForge CLI
ML DataForge" is a machine learning preprocessing tool designed to streamline the data preparation phase, saving valuable time for data scientists and analysts.

## Gist

- Given the database, the CLI provides various options to preprocess the data. 
- Options:
    - Data Description
    - Handling NULL Values
    - Encoding Categorical Data
    - Feature Scaling
- We can also **DOWNLOAD**:arrow_down: the modified dataset.
